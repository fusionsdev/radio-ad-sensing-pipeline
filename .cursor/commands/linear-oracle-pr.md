# Linear → Oracle → Agent PR

อ่าน `AGENTS.md`, `project-memory/workflows/git-safety-workflow.md`, และ `project-memory/workflows/linear-oracle-agent-workflow.md` ก่อนเริ่ม

1. ตรวจว่า branch คือ branch งานของ Linear / Oracle นี้
2. ตรวจว่า diff ยังจำกัดอยู่กับงานนี้
3. ถ้าไม่ตรง ให้หยุดแล้วถามก่อน
4. เปิด PR ไป `main`
5. ใช้ title ที่สั้นและตรงงาน
6. ใช้ body ที่สรุป:
   - summary
   - verification
   - scope
7. หลังสร้าง PR ให้รายงาน PR URL
