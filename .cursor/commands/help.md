# ช่วยเหลือ — เลือกคำสั่ง

ตอบภาษาไทย อธิบายสั้นๆ ว่า operator **ไม่ต้องจำศัพท์เทคนิค** มี 3 วิธี:

## วิธีที่ 1 — พูดตรงๆ (แนะนำ)
พิมพ์สิ่งที่อยากรู้เป็นภาษาคน เช่น "วันนี้ระบบ ok ไหม" agent จะใช้ skill ให้เอง **ไม่ต้องพิมพ์ /pipeline-ops**

## วิธีที่ 2 — กด `/` แล้วเลือก
ใน chat พิมพ์ `/` แล้วเลือก:
- `/status` — สถานะ pipeline
- `/cfpb` — ดู CFPB seeds
- `/cfpb-collect` — รวบรวมข้อมูล CFPB
- `/handoff` — สรุปก่อนปิด chat
- `/help` — หน้านี้

## วิธีที่ 3 — ก๊อปจากไฟล์
เปิด `plan/cursor-copy-paste.md` แล้วก๊อปบล็อกที่ต้องการ

แสดงตารางคำสั่ง `/` ทั้งหมดในโปรเจกต์นี้ให้ operator
